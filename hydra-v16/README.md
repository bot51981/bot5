## 🚀 Быстрый старт (5 минут)

### 1. Установка Python
- Скачайте Python 3.10+ с https://www.python.org/downloads/
- **Важно:** Установите флаг "Add Python to PATH"
- Перезагрузите компьютер

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Конфигурация API ключей

Отредактируйте файл `.env`:
```
BYBIT_API_KEY=ваш_ключ
BYBIT_API_SECRET=ваш_секрет
```

### 4. Запуск бота
```bash
python bot.py
```

### 5. (Опционально) Запустите сканер в отдельном терминале
```bash
python scanner_v3.py
```

## ⚙️ Основные настройки (config.json)

```json
{
  "trading": {
    "slot_size": 18.0,           // Размер позиции в USDT
    "entry_threshold": 0.75,     // % целевой прибыли
    "drop_threshold": 0.65,      // % падения для входа
    "panic_stop": 2.0,           // % стоп-лосс
    "use_dynamic_stops": true,   // Адаптивные стопы
    "atr_multiplier": 1.5        // Расстояние стопа (ATR)
  }
}
```

## 📊 Как это работает

### Сигнальный поток:
```
Цена падает ≥0.65%
        ↓
RSI < 30 (перепродано)?
        ↓
Цена > EMA9 > EMA21 (восходящий тренд)?
        ↓
MACD > Signal (импульс вверх)?
        ↓
Stochastic < 80 (не перекупленно)?
        ↓
✅ ВХОД В СДЕЛКУ
```

## 🔍 Диагностика проблем

### "Python не найден"
```
✅ Переустановите Python 3.10+
✅ Убедитесь, что "Add to PATH" отмечено
✅ Перезагрузитесь
```

### "Нет сигналов"
```
✅ Дайте боту время сканировать (5-10 минут)
✅ Снизьте drop_threshold в config.json
✅ Запустите сканер отдельно (run_scanner.bat)
```

### "Слишком много ложных сигналов"
```
✅ Увеличьте min_signal_score: 2 → 3
✅ Снизьте rsi_oversold: 30 → 25
✅ Увеличьте drop_threshold: 0.65 → 1.0
```

## 📞 Часто задаваемые вопросы

**Q: Сколько времени нужно запускать сканер?**
A: Постоянно в фоне для лучших результатов. Сканер работает каждые 10 минут.

**Q: Какая минимальная начальная сумма?**
A: Тестируйте с 10-50 USDT. После проверки 2+ недель увеличивайте.

**Q: Нужна ли VPN?**
A: Нет, если вы находитесь вне Китая. Bybit работает глобально.

**Q: Можно ли редактировать параметры во время работы?**
A: Да! Отредактируйте config.json, он перезагружается автоматически.
